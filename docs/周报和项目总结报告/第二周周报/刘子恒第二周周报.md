**三、 遇到的问题与困难**

*   **技术问题：**
    *   智能助手需要同时兼顾多意图识别、连续追问、真实业务数据查询与隐私安全；单次输入包含多个问题时，原有单意图处理链路会遗漏后续问题。招聘 Agent 调用模型时还存在执行时间较长、前后端运行状态衔接复杂等问题。
    *   **已尝试的解决方案：** 通过拆分连续业务问题并串行执行、建立严格的请求与响应 Schema、校验模型结构化输出、限制结果上下文中的敏感值传递，并增加本地规则兜底；同时为 Agent 模型调用增加分阶段超时配置，完善 Runtime 事件和前端结果展示，并通过单元测试、接口测试与并发测试覆盖关键边界。
*   **非技术问题：**
    *   本周改动同时覆盖智能助手、招聘 Agent、候选人导入、测试和文档，涉及前后端多个模块，联调范围较大，提交之间存在一定依赖关系。
    *   **已采取的措施：** 按功能链路拆分任务，及时同步接口契约、架构说明与测试用例，并通过小步提交保留清晰的修改记录，降低多人协作时的合并和回归风险。
