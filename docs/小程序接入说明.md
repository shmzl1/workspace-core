# 小程序接入说明

## 定位

微信小程序是 Sprint 3 规划中的员工端入口，仅提供员工简单能力。

## 一期范围

- 员工首页。
- 签到、签退。
- 今日考勤状态。
- 本月考勤摘要。
- 年假余额。
- 本人薪资摘要。
- 制度查询。

## 接入方式

- 复用一套 FastAPI 后端。
- 使用演示账号或测试员工身份。
- 通过配置文件读取 API 地址。
- 局域网调试时使用笔记本局域网 IP，不使用 `localhost`。

## 明确不做

- 不接真实微信 OAuth。
- 不做招聘、排期、薪资预审、审计后台。
- 不做 HR 管理功能。

## 后续扩展点

真实微信登录的扩展点为账号绑定与服务端身份校验。
