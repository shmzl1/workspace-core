# 小程序接入说明

## 定位

微信小程序是 Sprint 3 规划中的员工端入口，仅提供员工简单能力。

当前目录为骨架，状态是“计划中”，不表示页面、请求或局域网联调已经完成。

## 一期范围

- 员工首页。
- 签到、签退。
- 今日考勤状态。
- 本月考勤摘要。
- 年假余额。
- 本人薪资摘要。
- 简单制度查询。

## 接入方式

- 复用一套 FastAPI 后端。
- 使用演示账号或测试员工身份。
- 通过配置文件读取 API 地址。
- 局域网调试地址由本地配置提供，不使用 `localhost`，也不在仓库中写死真实局域网 IP。
- 只能访问当前登录员工本人授权数据，不在小程序内计算考勤、年假、薪资或 Agent 结果。

## 明确不做

- 不接真实微信 OAuth。
- 不做企业招聘目标、候选人评分、多 Agent 招聘决策。
- 不做面试排期、面试评价或招聘报告。
- 不做薪资预审、薪资确认、审计后台或 Gradio。
- 不做其他 HR 管理功能。

## 后续扩展点

真实微信登录的扩展点为账号绑定与服务端身份校验。
