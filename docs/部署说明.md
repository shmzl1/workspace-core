# 部署说明

本文记录计划部署方式，不表示系统已经完成部署。

## 计划部署方式

- FastAPI 后端运行在笔记本。
- `APP_HOST=0.0.0.0`，便于局域网设备访问。
- Vue Web 管理端、小程序员工端和 Gradio 内部调试台共享同一套 API。
- Docker Compose、Nginx、PostgreSQL 是 Sprint 3 收口目标。

## 局域网访问

- Web 本机调试可使用 `localhost`。
- 小程序不能使用 `localhost`。
- 小程序局域网 API 地址通过配置模板设置为笔记本局域网 IP。
- 手机与笔记本需要处于同一局域网。

## 后续真实微信发布要求

真实微信发布时需要处理：

- HTTPS。
- 合法域名。
- 微信 OAuth。
- 服务端身份校验。

当前阶段不做公网生产部署。
