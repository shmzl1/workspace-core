# 考勤与薪资预审规则

## 考勤规则

- 每人每天最多签到一次。
- 每人每天最多签退一次。
- 未签到不能签退。
- 仅做时间签到签退。
- 不做位置、人脸、Wi-Fi、蓝牙验证。

## 考勤状态

- 正常
- 迟到
- 早退
- 缺勤
- 无薪请假
- 已批准年假

已批准年假不扣基础工资。缺勤、无薪请假、迟到、早退可以进入薪资预审。

## 薪资预审公式

```text
应发工资预览
= 基础工资
+ 绩效奖金
+ 交通补贴
+ 餐补
- 缺勤扣款
- 迟到扣款
- 早退扣款
- 无薪请假扣款
```

## 薪资规则

```text
缺勤扣款 = 基础工资 / 标准工作日 * 缺勤天数
无薪请假扣款 = 基础工资 / 标准工作日 * 无薪请假天数
```

- 迟到与早退按次数扣款。
- 绩效奖金、交通补贴、餐补由 HR 录入。
- 每项扣款必须保留考勤来源、规则依据和计算过程。

## 权限与确认

- 规则引擎负责计算。
- AI 只负责解释和审查建议。
- HR 最终确认。
- Agent 不得确认工资、修改工资、删除扣款或写入已确认薪资。
- 所有敏感操作必须审计。
