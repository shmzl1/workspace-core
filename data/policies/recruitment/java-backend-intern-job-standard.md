# Java 后端开发实习生岗位标准

本文件为 TalentFlow 课程演示使用的虚构岗位标准，不包含真实企业或个人数据。

## 岗位信息

- 岗位编号：JOB-JAVA-INTERN-001
- 岗位名称：Java 后端开发实习生
- 部门：研发部
- 最低经验要求：0 个月，可接受在校项目经历

## 必备能力

- 掌握 Java 基础语法、集合、异常处理和常见面向对象设计。
- 能够编写并解释基础 SQL，理解索引、事务和关系模型。
- 理解 REST API 的资源设计、状态码和请求响应结构。
- 至少提供一段可核查的课程项目、个人项目或实习项目经历。

## 加分项

- 使用过 Spring Boot 构建后端服务。
- 使用过 PostgreSQL，并能说明表结构或查询优化思路。
- 了解自动化测试、Git 协作和接口文档维护。

所有能力判断都需要对应简历或面试证据。岗位标准只用于辅助筛选，最终决定由 HR 和业务面试官完成。
