# miniprogram 小程序约束

## 覆盖范围

本文件约束微信小程序员工端。

## 一期范围

小程序只提供员工端简单能力：

- 员工首页
- 签到
- 签退
- 今日考勤状态
- 本月考勤摘要
- 年假余额
- 本人薪资摘要
- 制度查询

## 架构规则

- 复用同一套 FastAPI 后端。
- 使用演示账号或测试员工身份。
- API 地址通过 `config/env.example.js` 派生本地配置。
- 小程序在局域网调试时不能使用 `localhost`，应使用笔记本局域网 IP。
- 只调用统一 FastAPI API。
- API 地址必须从配置读取。

## 明确不做

- 不接真实微信 OAuth。
- 不做招聘、候选人、排期、薪资预审、审计后台。
- 不缓存敏感完整薪资字段和密钥。
- 不做 GPS、Wi-Fi、人脸、蓝牙等位置考勤。
- 后续真实微信登录扩展点为账号绑定与服务端身份校验。
