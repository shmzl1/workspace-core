# miniprogram 小程序约束

## 覆盖范围

本文件约束微信小程序员工端。

## 一期范围

小程序只提供员工端简单能力：

- 员工首页
- 签到
- 签退
- 今日考勤状态
- 本月考勤摘要
- 年假余额
- 本人薪资摘要
- 制度查询

## 架构规则

- 复用同一套 FastAPI 后端。
- Vue Web 管理端、微信小程序员工端、Gradio 内部调试台共享同一套 FastAPI 后端。
- 后端采用模块化单体，不拆微服务，不新增第二套后端。
- 使用演示账号或测试员工身份。
- API 地址通过 `config/env.example.js` 派生本地配置。
- 小程序在局域网调试时不能使用 `localhost`；地址必须由本地配置提供，不得写死真实局域网 IP。
- 只调用统一 FastAPI API。
- API 地址必须从配置读取。
- 不在小程序内计算考勤、年假、薪资或 Agent 结果。
- 只能访问当前登录员工本人授权数据。

## 明确不做

- 不接真实微信 OAuth。
- 不做企业招聘目标、候选人评分、多 Agent 招聘决策、面试排期、面试评价、招聘报告、薪资预审、薪资确认、审计后台或 Gradio。
- 不缓存敏感完整薪资字段和密钥。
- 不做 GPS、Wi-Fi、人脸、蓝牙等位置考勤。
- 后续真实微信登录扩展点为账号绑定与服务端身份校验。
- 不直接访问数据库、禁飞区或底层算法。
- 不引入 Redis、Celery、RabbitMQ、Kubernetes 或第二套后端相关目录。
- 不伪造 Agent 日志，不展示模型隐藏思维链。
- 目录调整不得删除现有骨架或兼容路径。
- 默认不运行 npm、Docker、Git、构建或启动命令，除非用户明确要求。
